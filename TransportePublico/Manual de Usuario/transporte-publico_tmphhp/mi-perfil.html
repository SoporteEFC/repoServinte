Ventana Configuración de Usuario

Con este formulario, se puede modificar la información actual del usuario activo o con el cual se ingresó al sistema. Por ejemplo para cambiar la contraseña del usuario actual, coloque la contraseña actual en el campo de texto Password Anterior, seguidamente coloque la nueva contraseña en los campos de texto Password y Confirmar Password. Finalmente para confirmar los cambios de clic en el botón Guardar.
